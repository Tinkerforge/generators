Tinkerforge MATLAB/Octave Bindings
==================================

The MATLAB/Octave bindings allow you to control Tinkerforge Bricks and
Bricklets from your MATLAB/Octave scripts. This ZIP file contains:

 matlab/Tinkerforge.jar -- precompiled Java library for MATLAB
 matlab/source/         -- source code of Tinkerforge.jar for MATLAB
 matlab/examples/       -- MATLAB examples for every Brick and Bricklet
 octave/Tinkerforge.jar -- precompiled Java library for Octave
 octave/source/         -- source code of Tinkerforge.jar for Octave
 octave/examples/       -- Octave examples for every Brick and Bricklet

For more information about the MATLAB/Octave bindings (including setup
instructions) go to:

 https://www.tinkerforge.com/en/doc/Software/API_Bindings_MATLAB.html (English)
 https://www.tinkerforge.com/de/doc/Software/API_Bindings_MATLAB.html (German)
