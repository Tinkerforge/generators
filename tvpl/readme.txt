Tinkerforge Visual Programming Language (TVPL) Bindings
=======================================================

The TVPL bindings allow you to control Tinkerforge Bricks and Bricklets from
your TVPL programs. This ZIP file contains:

 source/   -- source code of the bindings
 examples/ -- examples for every Brick and Bricklet

For more information about the TVPL bindings (including setup instructions)
go to:

 http://www.tinkerforge.com/en/doc/Software/API_Bindings_TVPL.html (English)
 http://www.tinkerforge.com/de/doc/Software/API_Bindings_TVPL.html (German)

The TVPL bindings are also available als on online editor:

 http://www.tinkerforge.com/en/tvpl/editor.html (English)
 http://www.tinkerforge.com/de/tvpl/editor.html (German)
