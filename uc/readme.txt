Tinkerforge C/C++ Bindings for Microcontrollers
===============================================

The C/C++ bindings for microcontrollers allow you to control
Tinkerforge Bricks and Bricklets from
your C/C++ programs running on microcontrollers. This ZIP file contains:

 source/   -- source code of the bindings
 examples/ -- examples for every Brick and Bricklet

For more information about the C/C++ bindings (including setup instructions)
go to:

 https://www.tinkerforge.com/en/doc/Software/API_Bindings_uC.html (English)
 https://www.tinkerforge.com/de/doc/Software/API_Bindings_uC.html (German)
