Tinkerforge PHP Bindings
========================

The PHP bindings allow you to control Tinkerforge Bricks and Bricklets from
your PHP program. The Tinkerforge.tgz file is a PEAR package that can be
installed with the pear tool. The source/ directory contains the source code
of this PEAR package and the examples/ directory contains PHP examples for
every Brick and Bricklet. For more information about the PHP bindings go to:

 http://www.tinkerforge.com/en/doc/Software/API_Bindings_PHP.html (English)
 http://www.tinkerforge.com/de/doc/Software/API_Bindings_PHP.html (German)
