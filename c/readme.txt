Tinkerforge C/C++ Bindings
==========================

The C/C++ bindings allow you to control Tinkerforge Bricks and Bricklets from
your C/C++ program. The source/ directory contains the source code of the
bindings and the examples/ directory contains C/C++ examples for every Brick
and Bricklet. For more information about the C/C++ bindings go to:

 http://www.tinkerforge.com/en/doc/Software/API_Bindings_C.html (English)
 http://www.tinkerforge.com/de/doc/Software/API_Bindings_C.html (German)
