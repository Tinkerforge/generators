Tinkerforge Java Bindings
=========================

The Java bindings allow you to control Tinkerforge Bricks and Bricklets from
your Java program. The Tinkerforge.jar file is a precompiled Java library.
The source/ directory contains the source code of this Java library and the
examples/ directory contains Java examples for every Brick and Bricklet. For
more information about the Java bindings go to:

 http://www.tinkerforge.com/en/doc/Software/API_Bindings_Java.html (English)
 http://www.tinkerforge.com/de/doc/Software/API_Bindings_Java.html (German)

The Java bindings are also available from the Maven Central Repository:

 http://search.maven.org/#browse%7C1012008284
