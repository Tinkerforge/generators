Tinkerforge LabVIEW Bindings
============================

The LabVIEW bindings allow you to control Tinkerforge Bricks and Bricklets from
your LabVIEW program (Windows only). The Tinkerforge.dll file is a precompiled
.NET library. The source/ directory contains the source code of the .NET library
and the examples/ directory contains LabVIEW examples for every Brick and
Bricklet. For more information about the LabVIEW bindings go to:

 http://www.tinkerforge.com/en/doc/Software/API_Bindings_LabVIEW.html (English)
 http://www.tinkerforge.com/de/doc/Software/API_Bindings_LabVIEW.html (German)
