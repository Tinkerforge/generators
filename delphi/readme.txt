Tinkerforge Delphi/Lazarus Bindings
===================================

The Delphi/Lazarus bindings allow you to control Tinkerforge Bricks and
Bricklets from your Delphi/Lazarus program. The source/ directory contains the
source code of the bindings and the examples/ directory contains Delphi/Lazarus
examples for every Brick and Bricklet. For more information about the
Delphi/Lazarus bindings go to:

 http://www.tinkerforge.com/en/doc/Software/API_Bindings_Delphi.html (English)
 http://www.tinkerforge.com/de/doc/Software/API_Bindings_Delphi.html (German)
