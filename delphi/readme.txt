Tinkerforge Delphi/Lazarus Bindings
===================================

The Delphi/Lazarus bindings allow you to control Tinkerforge Bricks and
Bricklets from your Delphi/Lazarus programs. This ZIP file contains:

 source/   -- source code of the bindings
 examples/ -- examples for every Brick and Bricklet

For more information about the Delphi/Lazarus bindings (including setup
instructions) go to:

 https://www.tinkerforge.com/en/doc/Software/API_Bindings_Delphi.html (English)
 https://www.tinkerforge.com/de/doc/Software/API_Bindings_Delphi.html (German)
