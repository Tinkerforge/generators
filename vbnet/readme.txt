Tinkerforge Visual Basic .NET Bindings
======================================

The  Visual Basic .NET bindings allow you to control Tinkerforge Bricks and
Bricklets from your Visual Basic .NET program. The Tinkerforge.dll is a
precompiled .NET library. The source/ directory contains the source code of
this .NET library and the examples/ directory contains  Visual Basic .NET
examples for every Brick and Bricklet. For more information about the Visual
Basic .NET bindings go to:

 http://www.tinkerforge.com/en/doc/Software/API_Bindings_VBNET.html (English)
 http://www.tinkerforge.com/de/doc/Software/API_Bindings_VBNET.html (German)
